# Multi-Platform OTT Recommendation System

A Streamlit-based recommendation system that helps users discover movies and shows across multiple OTT platforms.

## Features
- Search movies and shows
- Platform-based filtering
- Type-based filtering
- Watchlist support
- Recommendation system using TF-IDF and cosine similarity

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn

## Dataset Files
- netflix_titles.csv
- disney_plus_titles.csv
- amazon_prime_titles.csv

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
